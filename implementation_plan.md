# 実装計画: Google AI Studio ランダムプロンプト生成器

この計画では、Node.jsを使用してGoogle AI Studio (Gemini API) にリクエストを送信するスクリプトを作成します。プロンプトは複数の要素をランダムに組み合わせて生成されます。

## 1. プロジェクトセットアップ
- `package.json` の初期化
- 必要なパッケージのインストール:
    - `@google/generative-ai`: Google AI Studio SDK
    - `dotenv`: 環境変数管理 (.envファイル用)

## 2. 環境設定
- `.env` ファイルを作成し、APIキー (`GEMINI_API_KEY`) を設定できるようにする。
- `.gitignore` に `.env` と `node_modules` を追加して、機密情報がコミットされないようにする。

## 3. スクリプト実装 (index.js)
- **プロンプト部品の定義**:
    - いくつかの配列を用意します（例：`subjects`（主語/テーマ）, `actions`（アクション）, `styles`（スタイル/口調））。
- **ランダム生成ロジック**:
    - 各配列からランダムに要素を選択し、1つのプロンプト文字列を作成する関数を実装します。
- **Gemini API呼び出し**:
    - 生成されたプロンプトを使用して `gemini-1.5-flash` (または `pro`) モデルにリクエストを送信します。
- **結果の出力**:
    - 生成されたテキストをコンソールに出力します。

## 4. 実行方法
- ユーザーに `API Key` の取得方法と設定方法を案内します。
- `node index.js` で実行できることを確認します。
